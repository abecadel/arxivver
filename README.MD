# arxivver

arxiv.com api feed reader for android 


## Build
run: gradlew clean build

apk is in app/build/outputs/apk/debug/app-debug.apk

## Build JavaDocs
run: gradlew createJavadocs

JavaDocs are in app/build/docs/javadoc/index.html
